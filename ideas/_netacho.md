## 🌱 興味関心リスト ＝ バックボーン／持ち札（2026-06-15 Takahiro投下）

> **これらは記事の"主軸"ではなく、バックボーン**（語りの具体・たとえ・声・説得力）。
> 一個一個を主軸にすると具体的すぎて間口が狭まる（ファンブログ化）。広い主軸テーマの中に**味付けとして織り込む**のが基本。たまに軽い単体エッセイにするのはアリ。
> 配合の原則: **主軸は広く（共感で開かせる）／バックボーンで具体を効かせる（"あなたの記事"にする）**。

- **自分を整える・律する系**（柱候補★）: 断酒 / 筋トレ / ランニング / 瞑想 / スマホ断ち / 自分をコントロールすること / 成長すること
- **さすらい・未知系**（柱候補★／上と対になる）: さすらう（目的なくうろうろ） / 遠くに行くこと / 旅行 / 飛行機 / 海 / 温泉 / 星空 / 新しいこと・新しいもの・未知のもの / 水曜どうでしょう・大泉洋
- **思想・議論系**（論考の土台）: 建設的な議論 / 哲学 / アナーキーでリベラルな考え方 / 神仏・神社仏閣 / 歴史
- **テック系**: IT技術 / AI / 新しいテクノロジー
- **漫画**: 藤田和日郎 / 皆川亮二 / 葬送のフリーレン
- **乗り物・機械系**: クルマ / バイク（ワインディングロード）/ 車いじり
- **食・グルメ系**（サイゼ路線で量産可）: ピザ / うなぎ / 蕎麦 / カレー / あんこ / エクレア / 豆腐類 / コーヒー / 料理
- **アウトドア・身体系**: キャンプ / 釣り / スノーボード / サッカー / 海
- **表現・趣味系**: 音楽（ロックンロール・クロマニョンズ）/ ギター / 写真 / 美術鑑賞 / ゲーム（ファミコン）
- **その他の芯**: うさぎ / 掃除 / イタリア / 水族館 / 英語 / サプライズ・人を驚かすこと

### 💡 並べて見えた"芯"（メモ）
- **二面性が"声"になる**: 「自分を律する（断酒・筋トレ・瞑想）」⇄「目的なくさすらう（うろうろ・旅・アナーキー）」。この相反を併せ持つのが、このアカウントの声＝バックボーンの芯。主軸テーマを貫く通奏低音として使える。
- 「自分を整える系」は 断酒シリーズを内包できる広い主軸。
- 食・グルメはサイゼと同じ"軽い実体験エッセイ"で手数を稼げる（ここは持ち札が主軸寄りになる例外）。
