## この章のまとめ

### note の概要
note株式会社が運営する文章投稿プラットフォーム（2024年で10周年）。誰でも文章を投稿し値段を付けて売れる。「個人の文章メディア」として機能し、一般人の文章も多数販売されている。

### note 収益化の手段（4つ）
1. 有料記事
2. 有料マガジン
3. 定期購読マガジン／メンバーシップ
4. サポート（投げ銭）

基本は「有料記事」の単体販売。これができないと他も難しい。

### 公式情報の活用
note社の公式情報・セミナーが収益化に役立つため活用が推奨される。

### 収益化の具体的方法（フリーミアム）
1. 自分の過去の「実体験」をコンテンツにまとめる
2. 試験的に「無料記事」を書いて需要の有無を見極める
3. 「9割無料」の記事を書き、最後の1割のオマケ部分だけを低単価で売る
4. 「1割だけ無料」の有料記事を書き、単価を上げて売る
5. いくつかのトピックで 4 を繰り返す

重要なのは「無料部分」が説得力を持ち、「有料部分を読みたい」価値を伝えること。

---

## 副業手段としての note の優位性
- 初期投資・ランニングコストがゼロ。赤字の心配がない。
- プレッシャー/ストレスが少なく、自分のペースで続けられる。
- 在庫管理・顧客対応が不要。納期・クライアントのプレッシャーがない。

### ブログとの比較
- note は広告収益モデルに依存せず、純粋に文章の価値で収益化。
- ブログは SEO・広告主配慮が必要で、文章の質より広告収益が優先されがち。

### プラットフォームとしての強み
note 自体が SEO に強く回遊率が高い。新規でも質の高い記事なら多くの読者に届く。

---

## HARM の法則に囚われてはいけない
有料note成功例の「家庭料理ノウハウ」「観光案内」等は HARM（健康・お金・夢・人間関係）に直接含まれないが、紛れもなく素人が書いてもお金になるテーマ。

**ポイント: 根源的な悩みか否かではなく、「リアルで生々しい体験とそこから生まれた感情」が文章中に内在しているか。**

## 有料 note 記事のテーマ例（20案・アイデアバンク）

### 個人的な出来事を語るエッセイ
- 父親との軋轢と、最近になって突然変わった関係性
- お世話になった先生が亡くなったとき、ふと思い出した高校時代のこと
- イジメと引きこもりを経験した私が考える、コミュニケーションの本質

### 悩み苦しみながらスキルを身に付けた経験
- 留学経験なしで海外に飛び出した私の英語習得方法
- 勉強が苦手な人でも楽しく学べる統計学の基礎知識
- 難関大学の受験を突破するまでに私がやったこと
- ベンチプレス200kgを上げられるようになる最短の方法

### 家族や友人に喜ばれたグルメと料理のノウハウ
- 簡単な材料で誰でも美味しく作れる最強パスタのレシピ公開
- 予算別：とびっきり美味しいのに空いている都内の穴場レストラン20選
- 20回以上通って完成させた、最新版：韓国ソウル市内のカフェマップ

### 自分の足で手に入れた旅行・観光の現地情報
- ガイドブックに載っていないマイナー観光地でめちゃくちゃ苦労した話
- 夏の暑い日に小さな子連れでもストレスなく楽しめるお出かけスポット
- アウトドア好きが教える、東京から日帰りで行ける快適なキャンプ地

### 長年の会社員の仕事で培った知識と経験
- 営業職としてトップに立つための姿勢と習慣づくり
- 40代後半の転職活動を通して得た、重要な気づき
- 子育てに悩むワーママが実践している時間管理術

### 膨大な努力の末に体得した個人的ノウハウ
- 1年間で100冊の本を読破するための私なりの読書術
- おもちゃを使わずに子どもの創造力を育てるスマートな遊び方
- ガーデニング初心者が知っておくべき基礎知識
- 週末DIYで楽しみながら作る、簡単でおしゃれな家具

いずれも HARM法則とは直接関係ないが、「リアルで生々しい体験とそこから生まれた感情」が内在している。

## note の禁止事項（クリエイター規約「9. 禁止事項」より抜粋）
1. 盗作・剽窃など他者の著作権等を侵害するもの
2. 財産権・商標権等の知的財産権、肖像権、名誉、プライバシー等を侵害するもの
3. 詐欺やそのおそれがあるもの
4. 性的な音声・画像・動画
5. わいせつ的・暴力的、その他過度の不快感を及ぼすおそれのあるもの
6. 差別につながる民族・宗教・人種・性別・年齢等に関するもの
7. 自殺・自傷・違法/脱法薬物使用等を勧誘・誘発・助長するおそれのあるもの
8. マルチ商法等、ユーザーに不利益をもたらすと判断される情報商材
9. 株式の銘柄推奨その他金融商品取引法に抵触するもの
10. 「必ずもうかる」等、著しい誤解を招く表現を用いたもの
11. コンピュータウイルス等、有害なプログラムを含むもの
12. オンラインゲームのアカウント・アイテム・通貨・仮想通貨等を譲渡しようとするもの
13. 景表法・薬機法・医療法その他広告関連法令に違反する/おそれのあるもの
14. 特定の個人・グループ・組織になりすますもの
15. 情報商材の宣伝に直接/間接的に利用するもの
16. 未成年者を犯罪行為等に勧誘するもの
17. 法令に違反するもの
18. 公序良俗に反するもの
19. 前各号の内容が掲載されたサイトへのリンクがあるもの
20. その他、前各号に準じる不適切なもの

## 特に気を付けること
- 他人の作品のパクリ・無断転載をしない（引用は著作権法のルールを守る）
- 18禁画像、差別、違法行為助長を書かない
- 「誰でも確実に儲かる」「誰でも簡単に10キロ痩せる」等の誤解を生む表現をしない
- 「100部限定で値下げ」と告知して実際は100部以上売る等の不当表示をしない

規約外で執筆時に注意すべき2点:
- **会社員の仕事で得た知識をそのまま note に書くのはやめた方が良い**
- **注目狙いで過激な内容を記事にするのはやめた方が良い**

## 副業の第一歩は「何を売るか」
- 購買は「欲しい」と思わせるのが前提。興味を引けない商品は売れない。

### 情報の希少性は重要ではない
- 多くの人が持つ情報でも稼げる。**実体験ベースの一次情報**が重要。

### 「誰が言うか」が大事
- 情報そのものより、込められたストーリーや感情に読者は価値を見出す。

### 競争に勝つ必要はない
- ナンバーワン/オンリーワンでなくてよい。「他人の体験を追体験したい」と思わせるのが鍵。

### 規約を守る
- 著作権侵害・誤解表現を避ける。企業秘密の販売・炎上狙いの過激記事は控える。

## note 記事を売るためのステップ
1. 過去の「実体験」をコンテンツにまとめる
2. 試験的に「無料記事」で需要を見極める
3. 「9割無料」記事の最後1割を低単価で売る
4. 「1割だけ無料」の有料記事を単価を上げて売る
5. いくつかのトピックで 4 を繰り返す

### 無料→有料の具体ステップ
- 「無料部分」だけでも十分な価値と説得力がある
- 「有料部分を読みたい」期待が伝わる
- 「無料」から「有料」への導線がスムーズ
- 「有料でも読む価値があった」満足感がある

### 読まれる無料記事の特徴
- タイトルは読者の興味を引くものに
- ハッシュタグを適切に付け、他クリエイター記事からの流入を狙う

### 失敗例・注意点
- サムネイル画像の作り込みは逆効果
- note の応募企画に無理に乗らない
- 他人の note への「スキ」「フォロー」周りは効果が薄い

## 有料 note の書き方の基本
- 文字数は**最低3,000字以上**を目安に
- **文字数と相場価格には相関がある**ので注意

### 読まれやすい文章のテクニック
- 段落・改行・箇条書きで読みやすく
- 「ここでしか読めないもの」を求める読者へ、一般論でなく個人的体験・思いを重視

### 結論は最初に書かない
- 無料パートと有料パートの区切りを工夫する
- **目次は無料パートの最後に設置する**（有料パートの最初ではない）

### 読了率を意識する
- 最後まで読んでもらうことが重要
- 文章のリズム、続きを読みたくなる工夫

### 読者の期待値コントロール
- 読む前に内容の範囲・対象を明確に
- 課金前に、解決できる問題/できない問題を伝える

## note 副業の本質
- 過去の経験が価値あるものと評価され、自己実現につながる
- 会社員でも手軽に始められるクリエイターエコノミーの一翼

### AI 活用
- ChatGPT 等の生成AIが執筆をサポート。効率的に記事作成が可能。note社もAI技術を導入。

### note の自由度
- 課金バリアに守られ、忖度のない質の高いコンテンツを表現できる

### 可能性
- 最初は月5万円でも、その先の可能性は無限大
- 「リアルで生々しい実体験のストーリー」は他の誰かの人生を救う

## 例文 B（無料/有料の区切りテクニックの実例）

> こんにちは、安齋響市です。
> 今日は、転職活動の際に入ってはいけない会社の見分け方について紹介します。
> 入社後に「こんなはずじゃなかった」と思わないように、事前に気を付けるべきポイント、企業に確認しておくべき質問内容などをまとめます。
> この記事をよく読んで実行すれば、転職前後のミスマッチをできる限り減らせるはずです。
> 最初に、結論を言いましょう。
> 入社してはいけない会社の見分け方とは、主に次の3つの項目です。
> （ここから先は有料部分）

### 目次（無料パート末尾に配置）
- 見分け方①：社内の〇〇〇〇の関係性を探る
- 見分け方②：入社前に絶対にしておくべき質問
- 見分け方③：意外と見落としがちなポイント
- この方法でもまだ不信感があるときは
- 中途入社前後のミスマッチをどう捉えるべきか
- おわりに

`3,824字／2画像 / この記事のみ：300円`

**学び**: 結論を無料パートで「予告」し詳細を有料に置く。目次は「有料パートの最初」ではなく「無料パートの最後」に入れると課金が増える。
